맨해튼 거리 기준임에도 불구하고 BFS로 하려고 해서 크게 시간이 끌렸다

맨해튼 거리일때 BFS를 하면 문제점은

산타들에 둘러싸여진 칸이 있을때

맨해턴 거리 기준으로는 그곳이 정답칸임에도 visited 값이 0 이어서 도달하지못하므로 문제이다

간단하게 앞으로 맨해튼 거리 기준일때는 BFS 를 사용하지않는다로 기억하자

또한 격자맵 바깥으로 차여서 추방당했음에도 바로 return 하지않아서 원래 추방되어진 산타와 부딪혀 die count가 올라가 게임이

일찍 종료되는 버그도 있었다

테케실제값에도 정답값보다 전반적으로 적게 나왔었다
