https://www.acmicpc.net/problem/1012

DFS 유형을 새로 알게되었다.
단순히 cnt를 꾸준히 더해 길이 값을 더해나가는 방식말고도
해당 유형처럼 고립된 섬이 몇개인지 구하는 방식도 DFS로 구한다는 것을 알게되었다.
원리는 간단하다. DFS 호출 전에 cnt를 더하면 DFS에서 재귀함수를 반복하며 그 섬을 모두 visited 처리하기 때문에 가능하다.

또한 fill 함수로 처음 실수를 했는데
```cpp
fill(&adj[0][0], &adj[0][0]+51*51, 0);
fill(&visited[0][0], &visited[0][0]+51*51, 0);
```
이 fill 함수로 테스크케이스마다 초기화를 한다면 
반드시 맵 사이즈가 51*51여야 한다
만약 맵사이즈가 100*100인데 저 2줄로 초기화를 하면

100 * 100 배열에서 adj[51][0]은 5100번째 칸에 해당하므로 초기화되지않는다
반드시 fill 함수에서 초기화 바이트를 선언할때 맵 사이즈를 고려해야한다.
