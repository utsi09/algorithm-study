# 내생각

조합 구현법을 까먹었어서 예전 코드를 보고 와서 풀었다
이제와서 보니 조합 구현법에도 백트래킹이 녹여져있는게 보였다

맵을 각 좌표를 페어로 받아서 vector에 저장한다
각 치킨집을 어떤 조합으로 구성할지를 다 인덱스를 기준으로 짜놓는다
그 인덱스를 기준으로 완탐을 돌린다 

그외엔 어려울 게 없는데 문제 이해도가 떨어져서 마지막에 헤멨다
각 집에서 가장 가까운 치킨집들을 짝으로 맺어서 그 거리들을 다 더해주는 문제인데
각 집에서 모든 치킨집까지의 거리를 다 더하고 조합인덱스중에서 가장 적은 값을 출력하고 있었다.
문제 이해가 가장 중요한 부분이라는걸 알게되었다
