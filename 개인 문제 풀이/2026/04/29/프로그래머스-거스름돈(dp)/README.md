솔루션 함수에 기능을 구현하는 방식이다

0을 만드는 경우의수는 1개이므로 초기화하고

가진 동전의 개수들로 반복돌리고

10만까지 정적할당한다음

목적 dp 인덱스보다 작은 coin들을 이용해서 해당 dp 값에 동전들값에 해당하는 dp들을 더해주면 된다

이부분은 이해하는데 꽤 오래걸렸는데

계단이 4층까지 가는 경우의수를 구하고 1,2,3 층을 이용할수잇으면

1층에서 3칸 올라가는경우, 2층에서 2칸 올라가는 경우 3층에서 1칸 올라가는 경우를 더한 값과 같다
